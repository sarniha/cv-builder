# 📄 CV Builder (React)

## 📌 Overview

This is a **modern CV/Resume Builder** built using **React (via CDN + Babel)**.
It allows users to **create, edit, and preview their resume in real time** with a clean and elegant UI.

The app follows a **live-editing approach**:

* Input details on the left panel
* Instantly see a formatted CV on the right panel

---

## ✨ Features

* 🧾 **Real-time CV Preview**
* 🧠 **Dynamic Form Handling (React Hooks)**
* 🎨 **Elegant UI with Custom CSS Design**
* ➕ Add / Edit / Remove:

  * Education entries
  * Work experience
* 🧩 Modular component-based architecture
* 📱 Responsive layout (works on smaller screens)

---

## 🛠️ Tech Stack

* **React 18 (CDN)**
* **JavaScript (ES6+)**
* **HTML5**
* **CSS3 (Custom Design System)**
* **Babel (for JSX support in browser)**

## ⚙️ How It Works

### 1. General Information

* Add your **name, email, phone**
* Toggle between edit and preview mode

### 2. Education Section

* Add multiple entries
* Save or edit anytime

### 3. Experience Section

* Add job roles, company, duration, and description

### 4. Live Preview

* Only saved entries are shown
* Clean CV layout updates instantly
